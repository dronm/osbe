3) Сервер событий.
<controller id="Order>
	<publicMethod id="get_make_list" cashable="TRUE>- Кэшируемый метод!
		<field id="date" required="TRUE cashKey="TRUE" /> - Это ключ!!! для очистки кэша!
		<dependantEvents id="Shipment.insert"/> это события, которые будут очищать кэш при совпадении ключей
		<dependantEvents id="Shipment.update"/>
		<dependantEvents id="Shipment.delete"/>
	</publicMethod>
</controller>

При вызове Order->get_make_list(date) должен быть проанализирован КЭШ! если есть отдать из кэша, нет сохранить в кэш
c=Order_Controller&f=get_make_list&date=2020-10-19 &v=ViewXML&t=
v=ViewXML - в ключе не участвует, удалять все view!!!
HASH1 =  (Controller + function + all parameters )
HASH2 = all other params
Файл кэша hash( c=Order_Controller&f=get_make_list&date=2020-10-19 ) = HASH1.HASH2
CRC-32 or Adler-32
For clearing hash use HASH1.*

Automatic script generation for clearing cash for Node js event server!
